感觉代码对算法7、8是进行了一部分修改和简化

首先是models.py

代码实现了两个方法：

  • get_model()

    获得模型，支持的模型包括 ResNet18、ResNet50、DenseNet121、AlexNet、VGG16、VGG19、Inception V3 和 GoogleNet。用户可以通过指定参数 name 来选择使用的模型，并可以通过指定参数 pretrained 来决定是否使用预训练权重。

  • model_norm(model_1, model_2)

    对model_1和model_2中每层参数的差求平方和后相加，最后对得到的平方和开平方，实现的

    中圈红部分(L2范数)。这里github代码实现中和书中给出的公式是不一样的。

然后看一下datasets.py

  • get_dataset(dir, name)：

    获取数据集并对数据集进行一些操作，返回训练数据集和评估数据集

    • transforms.RandomCrop(32, padding=4)

      • 随机裁剪

      • 在训练数据上随机裁剪出一块大小为32x32的图像

      • 如果图像的原始大小小于32x32，则不会进行裁剪

      • 为了避免在裁剪图像时丢失一些图像信息，可以通过设置padding参数对图像进行扩充

      • padding=4意味着在图像的四个边缘各增加4个像素

    • transforms.RandomHorizontalFlip()

      • 随机水平翻转
      • 随机地将图像水平翻转，从而对图像进行数据增强。水平翻转可以有效地扩大图像数据量，从而提高模型的泛化能力

    • transforms.ToTensor()

      • 为 PyTorch Tensor 对象

    • transforms.Normalize()

      • 对数据进行归一化
      • 将每个数据维度上的像素值归一化到一定的范围，从而帮助模型更好地学习数据的特征
